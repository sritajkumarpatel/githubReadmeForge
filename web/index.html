Multi-agent README generator

From a GitHub link to a polished README in under a minute.

githubReadmeForge uses a three-agent AI pipeline to scan your repo, identify gaps, and forge production-grade documentation — in any style, any language. No boilerplate, no fluff, no broken badges.

  • 5 battle-tested styles — reference, narrative, tutorial, showcase, or minimal. Modeled after the top 100 GitHub READMEs.
  • 5 LLM providers — Gemini, OpenAI, Claude, Ollama (local), or OpenCode. Your key stays in this tab.
  • Diff view + companion site — see exactly what changed and publish a beautiful HTML showcase alongside your README.
